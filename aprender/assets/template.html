About theme

This is a simple template created entirely in html and css. It is what we usually call a "three column fluid layout". For simplicity, everything is kept minimal.

We many times need layout (simple one like this) to create projects that provide server side functionalities. Projecs in which we do not concentrate on how things look!

This is the reason I created this template. Of course not a great thing! You can modify this template anyway you want. Add remove anything including my name from there. No issues!
